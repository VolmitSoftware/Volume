# Volume
A shared library across projects. Strives to accelerate plugin creation.

```xml
<repository>
    <id>volmit</id>
    <url>http://nexus.volmit.com/content/repositories/volmit</url>
</repository>
```

```xml
<dependency>
    <groupId>com.volmit</groupId>
    <artifactId>Volume</artifactId>
    <version>1.0</version>
</dependency>
```
